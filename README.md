# Description -

Memorize is a web application that helps you to create mind maps. Mind map is a technique to represent large information in diagrams & pictures. Using mind maps can be helpful for students while studying to remember lengthy information from subjects like History, Geography etc. It can help professionals as well as in scenarios like remembering a speech, summarizing terms & conditions, extracting meaningful information from text etc.
